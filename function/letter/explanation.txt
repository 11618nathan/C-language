
문자 함수 헤더 파일 선언 - #include <ctype.h>


문자 판단 함수
	- isalnum: 알파벳 또는 숫자인지 검사
	- isalpha: 알파벳 검사
	- iscntrl: 제어문 검사
	- isdigit: 숫자 검사
	- isgraph: 그래프 문자 검사
	- islower: 소문자 검사
	- isprint: 출력가능문자 검사
	- ispunct: 구두 문자 검사
	- isspace: 공백문자 검사
	- isupper: 대문자 검사
	- isxdigit: 16진수 문자 검사


문자 변환 함수
	- tolower: 소문자로 변경
	- toupper: 대문자로 변경
